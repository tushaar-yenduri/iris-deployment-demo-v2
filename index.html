1. Business Case

The goal of this project is to create a simple yet complete machine learning workflow â€” from understanding the business requirement to deploying a working web application. We use the classic Iris dataset to predict the species of a flower based on features such as sepal and petal dimensions.

2. Dataset Overview

The Iris dataset is publicly available on Kaggle. It contains 150 samples divided into three species: Iris Setosa, Iris Versicolor, and Iris Virginica. Each sample has four numeric features:

  • Sepal length (cm)
  • Sepal width (cm)
  • Petal length (cm)
  • Petal width (cm)

3. Data Preprocessing

  • Checked for missing or null values (none found)
  • Verified data types and ranges
  • Visualized feature relationships using Seaborn
  • Split the dataset into training and test sets (typically 80/20)

4. Model Selection â€“ Why Logistic Regression?

  • Simple and interpretable.
  • Works well on small, linearly separable datasets like Iris.
  • Computationally light and fast to train.
  • Handles multi-class classification via a one-vs-rest approach.

The trained model classifies new flower measurements into one of the three iris species. Typical accuracy exceeds 95%.
